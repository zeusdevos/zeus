## 🐛 Common Issues & Fixes

### ❌ "Failed to Attach" Error on Launch
**Cause:** Zeus opened before the Roblox game world fully loaded, or another process is blocking injection.
**Fix:** Ensure Roblox is fully in-game (character spawned), then click Attach in Zeus. If the issue persists, restart both Roblox and Zeus as Administrator.

---

### ❌ Windows Defender Removes Zeus Files
**Cause:** Heuristic false positive on the injector binary — standard behavior with all executors.
**Fix:** Add the Zeus Executor folder to Windows Defender Exclusions via `Windows Security → Virus & Threat Protection → Exclusions → Add a Folder`, then re-extract the archive.

---

### ❌ macOS Says "App Cannot Be Opened"
**Cause:** macOS Gatekeeper blocking an unnotarized developer binary.
**Fix:** Go to `System Settings → Privacy & Security`, scroll to the Security section, and click **Open Anyway** next to the Zeus X entry. You only need to do this once.

---

### ❌ APK Won't Install on Android
**Cause:** Install Unknown Apps permission not granted for your file manager app.
**Fix:** Go to `Settings → Apps → [Your File Manager] → Install Unknown Apps` and toggle it on, then retry the installation.

---

### ❌ Script Hub Fails to Load
**Cause:** No internet connection at launch, or the Script Hub CDN is temporarily unreachable.
**Fix:** Check your internet connection and relaunch Zeus. If the hub still won't load, disable your VPN if one is active — some VPN exit nodes block the Script Hub CDN.

---

### ❌ Auto-Execute Not Firing on Game Join
**Cause:** The auto-execute script slot is empty, or the feature wasn't enabled before joining the game.
**Fix:** Open Zeus settings, navigate to **Auto-Execute**, paste your script into the designated slot, and toggle the feature on. Save settings before launching Roblox.

---

### ❌ Zeus Crashes After a Few Minutes
**Cause:** Insufficient RAM or a conflicting background process consuming memory.
**Fix:** Close unused browser tabs and background applications. Ensure at least 4 GB RAM is free. On Windows, always run Zeus as Administrator to ensure proper memory access permissions.
